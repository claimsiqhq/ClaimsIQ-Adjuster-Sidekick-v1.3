🚀 TO BUILD THE APP:

1. Install dependencies (5 minutes):
   npm install expo-document-picker expo-sharing react-native-pdf \
     expo-image-manipulator @react-native-community/netinfo drizzle-orm \
     react-native-maps expo-location @sentry/react-native

2. Setup LiDAR in Xcode (10 minutes):
   - See LIDAR_XCODE_SETUP.md for detailed instructions
   - Add Swift files to Xcode project
   - Configure bridging header
   - Add ARKit/RealityKit frameworks

3. Prebuild native code (3 minutes):
   npx expo prebuild --clean

4. Install iOS pods (2 minutes):
   cd ios && pod install && cd ..

5. Deploy FNOL edge function (1 minute):
   npx supabase functions deploy fnol-extract

6. Build app (15-20 minutes):
   eas build --platform ios --profile preview

7. Install on iPhone and test!
